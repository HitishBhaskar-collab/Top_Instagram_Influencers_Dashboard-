# Top Instagram Influencers Dashboard
  This repository contains an interactive Power BI dashboard that visualizes insights from a dataset of top Instagram influencers. The dashboard helps users explore influencer performance through engagement metrics, follower statistics, and country-level trends.

# Features
  KPI cards for Total Followers, Total Likes, Average Engagement Rate, and Total Influencers
  Top 10 influencers ranked by score
  Followers vs Average Likes scatter analysis
  Country-wise engagement and total likes visualizations
  Engagement rate by rank trend analysis
  Interactive filters for Followers, Influence Score, Engagement Rate, and Country
  
# Files
  top_insta_influencers_data.csv – Dataset
  Output.png – Dashboard screenshot
  
# Tools Used
  Power BI
  CSV Dataset
  Data Visualization and Analytics

# Overview
  This dashboard is designed for marketers, analysts, and creators to understand influencer performance and make data-driven decisions in influencer marketing.
